# 中文绘图如何设置matplotlib

## 对于ubuntu系统

参考：https://blog.csdn.net/jeff_liu_sky_/article/details/54023745

## 对于windows

添加一句：

```python
import matplotlib
matplotlib.rc('font', family='simhei')
```

其中family直接指示字体名字就行，比如simhei

注意：如果不能正常显示，请更新matplotlib
